Data type

There are two basic types of data: quantitative, qualitative. Quantitative data is anything we can count, measure, or quantify with numberss. On the other hand, qualitative data represents any information in terms of language or description. We can not measure or count it. For example: the name of a person is qualitative data while the age of a person is quantitative data. We express quantitative data by using numbers, and qualitative data by using text or string.

Basic data type of programming

Apart from number and text, we use another basic types of data in programming languages that are boolean. Boolean has only two values, true and false.

Number type data

In programming, primarily we use three basic types of number: integer or whole number, decimal, and float.

Integer and decimal

All positive, negative whole numbers including 0 are integers. Briefly, all the whole numbers of the number line are integers. A decimal number is a real number that expresses fractions or whole numbers using a decimal point. In a decimal number, the decimal point separates the whole number part from the fractional part. For example, in the decimal number 3.14, "3" is the whole number part, and "0.14" is the fractional part. For example: 1,20, -30, 8,0 all are integers. On the other hand, 10.34, 10.67 are decimal numbers. Though there are few differences between decimal and float, for the bringing we can consider both as same.

String

In programming, a "string" is a way to handle text or qualitative data. It could be a single word, a sentence, or even a collection of paragraphs. To create a string, we wrap our text in quotation marks. For example: 


var greeting = "Hello"
var sentence = "This is a sample sentence."

we can combine strings by using the `+` operator:

 
var full_greeting = greeting + " " + sentence
//This would result in: "Hello This is a sample sentence."

Strings are just sequences of characters, and we can do basic operations like putting them together or taking parts of them. They are handy for dealing with any kind of text-related information in programming. In most programming languages, strings can be written inside either double quotation marks ("string" ) , single quotation marks ('string') , or triple quotation marks ('''string''' ) depending on the purpose of use and language syntax. Double quotes are commonly used, but using single quotes can be useful when dealing with strings that contain double quotes, and vice versa. Triple quotes are often used for multiline strings.

Boolean

Boolean refers to a data type that can have one of two values: true or false . Boolean values are often used in conditional statements and expressions to make decisions in computer programs.

Let's think about some examples:

  1. Is it sunny outside?
    • If it is sunny, we can say it's true.
    • If it's not sunny, we can say it's false.
  2. Do we have a pet?
    • If we have a pet, it's true.
    • If we don't have a pet, it's false.

Sometimes the true value represents 1 , and the false value represents 0 .

Boolean is like a digital switch that helps computers make decisions. It can be either "on" (true) or "off" (false). Think of it like answering questions with a "yes" or "no."

Uses of Boolean in Programming

Decision Making: Boolean helps programs decide what to do based on conditions. For example, if it's raining (true), bring an umbrella; otherwise (false), don't.

Comparison: It allows us to compare things, like whether one number is bigger than another. For instance, is 10 greater than 5? (true)

Logical Operation: Boolean helps express logical relationships. we can say, "I want ice cream AND it's a sunny day," meaning both conditions must be true.

Looping: In programming, Boolean is used to repeat tasks until a condition is no longer true. Like eating cookies while we're hungry (true).

Variable

A variable is like a storage box in which we can keep different types of information. Imagine it as a labelled container where we can store numbers, words, or other pieces of data. It's associated with a symbolic name and may have a memory address. The name allows referencing the data, and values can change during execution. To store a separate value, we need to use a separate variable. Abstractly, we can say a variable is a named storage location of any kind of data.

Declaring a Variable

In most programming languages, declaring a variable involves giving it a name and specifying the type of data it will hold. Here's a simple example in Javascript


//Declare a variable named 'age' and assign the value 25 to it
var age = 25

In this example, 'age' is the variable name, and '25' is the data assigned to it. The keyword "var" indicates that it’s a variable. There are some other conventions to declare a variable in Javascript. In some programming languages we need to specify the data type when declaring a variable. Let’s see an example from C programming language: 


int a = 8 //the keyword int indicates it will store integer type data 
float = 80.10 //the keyword float indicates it will store float or decimal type data 
char name = "my name" //the keyword char indicates that it will store string or text type data.

But in Javascript or Python we are not required to specify the data type when we declare a variable. These programming languages dynamically identify the data type of a variable.

Arithmetic Operators

Arithmetic operators are symbols used in programming to perform mathematical calculations on numerical data. They include:

  • Addition (+): Adds two values together.
  • Subtraction (-): Subtracts the right operand from the left operand.
  • Multiplication (*): Multiplies two values.
  • Division (/): Divides the left operand by the right operand.
  • Modulus (%):

    The modulus operator (%), when applied to the division of the left operand by the right operand, returns the remainder. For instance, consider the example: var a = 11 % 2 . In this case, the variable 'a' would be assigned the value 1. This implies that 'a' stores the remainder when 11 is divided by 2. The modulus operator is particularly useful for determining whether a given number is odd or even.

Basic Arithmetic Examples:

Addition: var total = 5 + 3; // total will be 8.

Subtraction: var difference = 10 - 4; // difference will be 6.

Multiplication: var product = 2 * 6; // product will be 12.

Division: var quotient = 8 / 2; // quotient will be 4.

Modulus: var remainder = 10 % 3; // remainder will be 1.
